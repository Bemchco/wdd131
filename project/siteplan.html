REMIS Online

REMIS: Redes, Equipo, Multiservicios, Informatica y Suministros

Description: This name represents the variety of services offered by the organization.

Domain available: remisonline.com.gt.

Site Purpose

A simple website that describes the company, shows the projects, presents sold projects examples, It would be something like a website to contact the company and would be like a Gallery with various projects displayed

Scenarios

  • What projects have we completed?
  • What is our experience and focus?
  • Where can I find contact information for a project?

Color Schema

The color palette for the site includes:

Header Color: Crimson: HEX #d72638ff;

Buttons: Celadon: HEX #a9e5bbff;

Primary Color: Licorice: HEX #0b0014ff;

Accent Color: Tigers eye: HEX #a85a00;

Background Color: White: HEX #ffffff;

Typography

Font 1: Arial - used for body text and paragraphs.

Font 2: Georgia - used for headings to give a classic and readable look.

Wireframe

Below are wireframe sketches for the homepage layout:

  • Mobile View: Single-column layout with stacked sections for logo, navigation, featured content, and footer.
  • Desktop View: Multi-column layout with a top navigation bar, sidebar for links, main content area, and footer.
